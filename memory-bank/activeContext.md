# Active Context

## تمرکز فعلی کار
- پیاده‌سازی ساختار اصلی ماژول تکمیل شده
- قابلیت ارسال پیام متنی اضافه شده
- قابلیت ارسال رسانه (عکس، ویدیو، صدا، سند) اضافه شده
- اعتبارسنجی شماره‌ها با کد کشور پیاده‌سازی شده
- در انتظار دریافت قابلیت‌های بیشتر (دریافت پیام، و غیره)

## تغییرات اخیر
- ایجاد ساختار کامل Package Laravel
- پیاده‌سازی WhatsAppService با مدیریت URL و API Key سراسری
- افزودن متد sendMedia() برای ارسال رسانه
- افزودن متد validatePhoneNumber() برای اعتبارسنجی شماره‌ها با کد کشور
- به‌روزرسانی sendMessage() برای اعتبارسنجی شماره‌ها
- ایجاد Service Provider و Facade
- ایجاد Controller و Routes برای API endpoints (send-message و send-media)
- ایجاد فایل Config با پشتیبانی از متغیرهای محیطی
- به‌روزرسانی README.md با مستندات کامل ارسال رسانه

## مراحل بعدی
1. تست ماژول در محیط Laravel
2. افزودن قابلیت‌های بیشتر (در صورت نیاز)
3. بهبود مدیریت خطاها

## تصمیمات فعال
- استفاده از ساختار استاندارد Laravel Package
- مدیریت URL و API Key از طریق Config (سراسری)
- استفاده از Facade برای دسترسی آسان
- پشتیبانی از Laravel 10.x و 11.x
