# Product Context: WhatsApp Module

## چرا این پروژه وجود دارد؟
این ماژول برای تسهیل ارسال پیام‌های واتساپ در پروژه‌های لاراول طراحی می‌شود.

## مشکلاتی که حل می‌کند
- نیاز به یک راه حل یکپارچه برای ارسال پیام واتساپ
- ساده‌سازی فرآیند اتصال به سرویس واتساپ
- قابلیت استفاده مجدد در پروژه‌های مختلف

## نحوه کار
- نصب از طریق Composer
- پیکربندی ساده
- استفاده از API endpoints برای ارسال پیام

## اهداف تجربه کاربری
- سادگی در استفاده
- مستندات کامل
- خطاهای واضح و قابل فهم
